# 🍕 Food Delivery Analytics Dashboard | Power BI

An end-to-end interactive Business Intelligence Dashboard built on a real-world food delivery dataset using **Power BI**, **DAX**, and **Power Query Editor**.

---

## 📊 Dashboard Overview

The dashboard consists of **6 interactive pages** with slicers, buttons, filters, and drill-through functionality:

| Page | Description |
|------|-------------|
| 🏠 Executive Overview | High-level KPIs — Total Orders, Revenue, Avg Order Value, Delivery Time, Revenue & Order Trends |
| 👥 Customer Analytics | New vs Repeat Customers, Sign-up Trends, Top 10 Customers, Orders by Customer Type |
| 🍽️ Restaurant's Performance | Revenue by Restaurant, Cost Bucket Analysis, Rating Trends, Rating wise Revenue |
| 🚚 Delivery and Operations | City-wise Delivery Time, Cancellation Analysis, Late Delivery %, Month Slicer |
| 💡 Advance Insights | Dynamic KPI selector, Revenue vs Rating vs Cost Bubble Chart, Q&A Visual |
| 🔍 Restaurant Details | Drill-through page — click any restaurant name to see its individual performance details |

## ✨ Key Features

- 🔘 **Interactive Buttons** — Navigate between pages using custom buttons
- 🔍 **Drill-Through** — Click on any restaurant in Executive Overview to drill through to its detailed performance page
- 🎛️ **Slicers & Filters** — Filter by Month, City, Online Orders, Book Table, and more
- 📐 **DAX Measures** — 10+ custom measures including Late Delivery %, Repeat Customer %, Revenue per Restaurant, and more
- 🧹 **Power Query** — Raw messy data cleaned, transformed, and modeled using Power Query Editor
- 💬 **Q&A Visual** — Natural language query support on Advance Insights page

---

## 💡 Key Business Insights

- 📦 **200K Total Orders** processed across Bangalore
- 💰 **₹164.91M Total Revenue** generated
- ⏱️ **44.55 mins** Average Delivery Time
- ⚠️ **48.12% Late Delivery Rate** — major operational challenge
- 🔁 **50.28% Repeat Customer Rate** — strong customer retention
- 🍽️ **Budget restaurants dominate** with 57.5% of total orders
- ⭐ **Average Restaurant Rating: 3.70/5**
- 📍 **BTM Layout** is the highest revenue generating area

## 🚀 How to Use

1. Download the `.pbix` file
2. Open with **Power BI Desktop** (free download from Microsoft)
3. Explore all 6 pages using the navigation buttons
4. Use slicers to filter by Month, City, and more
5. Click on any restaurant name in Executive overview to **drill through** to its detailed page
